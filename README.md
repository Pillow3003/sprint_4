# BooksCollector — Тестирование класса на Python

Этот проект предназначен для автоматизированного тестирования класса `BooksCollector`, который реализует функциональность по управлению коллекцией книг. Тесты написаны с использованием популярной библиотеки для тестирования — [pytest](https://pytest.org/).

---

## Описание

Класс `BooksCollector` предназначен для хранения информации о книгах, их жанрах, а также управления списком любимых книг. В рамках проекта реализованы и протестированы следующие методы:

- `add_new_book`: добавляет новую книгу в коллекцию, если имя валидное и не дублируется
- `set_book_genre`: задаёт жанр для указанной книги, при этом проверяются допустимые жанры и существование книги
- `get_book_genre`: возвращает жанр для указанной книги или `None`, если книга не найдена
- `get_books_with_specific_genre`: возвращает список книг заданного жанра
- `get_books_genre`: возвращает весь словарь с книгами и их жанрами
- `get_books_for_children`: возвращает список книг, подходящих для детей (например, не жанр "Ужасы")
- `add_book_in_favorites`: добавляет книгу в список избранных
- `delete_book_from_favorites`: удаляет книгу из списка избранных
- `get_list_of_favorites_books`: возвращает список всех книг, добавленных в избранное

---

## Детали тестирования

Класс `TestBooksCollector` содержит набор тестов, которые проверяют корректность работы методов. Каждое тестовое утверждение помогает убедиться, что:

- Методы работают при правильных вводных данных
- Обрабатывают некорректные или граничные случаи
- Не возникают неожиданные изменения или ошибки при выполнении операций

### Какие сценарии покрыты тестами:

- **Добавление новых книг**:
  - Валидация имени книги (длина, пустая строка, повторное добавление)
- **Управление жанрами**:
  - Обновление жанра существующей книги
  - Попытки установки недопустимого жанра
  - Обработка несуществующих книг
- **Получение информации**:
  - Получение жанра по названию
  - Получение списка книг определённого жанра
  - Получение всех книг
  - Получение книг для детей
- **Управление списком любимых книг**:
  - Добавление в избранное
  - Удаление из избранных
  - Получение текущего списка избранного
